# Proxies



## Forward proxy

forward proxy 通常与客服端的距离比服务器的距离更近，通常距离任何特定客户端一两个网格跳



## Reverse proxy

reverse proxy 反向代理是一种代理服务器，它离服务器比离客户端更近，对客户端完全透明
